Есть 3 способа добавить CSS в HTML
  • Подключить отдельный файл css в html коде страницы
  • Написать вначале самой страницы
  • Написать стиль в атрибуте style для каждого тега отдельно
Ссылкой подключен отдельный файл стилей task1.css
В отдельном теге стилей прописаны правила стилей для всех параграфов
Для одного параграфа прописаны отдельные стили в атрибуте style
